Datawhale动手学agent开发——Agent产品案例深度拆解

Datawhale动手学agent开发——Agent产品案例深度拆解

本次拆解的是David Bressler的AI产品formulabot，这款产品最初只是为了利用AI将指令转化为excel指令，但目前已经集成了数据分析，数据可视化和常见的文件转换等功能。

AI产品机会发现

创始人David原来是一家公司的数据分析主管。2022年时，ChatGPT在国外就有了一定的知名度，David接触到ChatGPT后就尝试运用到工作当中，众所周知，数据分析少不了要与Excel打交道，但遗憾的是，AI可以根据指令生成一些简单的Excel公式，对于复杂公式就不能达到令人满意的效果。

David尝试继续研究了当时的其他的AI工具，发现还没有能解决稳定生成Excel公式的产品，对大部分职场人士来说，Excel是他们的必需品，但说到精通，能写出复杂公式的恐怕就没有多少，工作时万一要用到复杂公式又来不及学，就成了广大职场人面临的共同难题。需求找到了，市场调研也误打误撞地做了，David觉得是时候做出一款Excel公式生成的产品了。

AI产品设计、开发与上线

由于并非技术出身，David虽有一个好的想法，却苦于无法实现，后来在不断的学习中，他发现了一个低代码建立网站的产品Bubble，同时他也意识到这个需求虽然重要，但对于技术人员实现起来并不难，一旦被人抢占先机，就只能扼腕叹息了。因此他快速搭建起来了网站的基本框架，实现核心功能：用户通过自然语言和AI交互，AI再转换为Excel公式。网站只有输入框，生成按钮，公式输出框和复制按钮四个基本组件，但符合AI产品用户体验的第一原则，让核心功能最快展现在用户眼前。

David的例子完美体现了二姐所说的AI产品实现的正确方式，产品要以用户为中心，不一定需要多么复杂的技术，重要的是满足用户的需求，初期的产品功能可能比较单一，但可以随着时间来逐步丰富。David并非技术人才，反而更容易跳出技术看出来AI产品是什么，没有陷入技术怪圈，这也是我们技术出身应该反思的，明明有技术，却不知道用来做什么。

网站上线后，formulabot 与 Cursor 一样，先在 Reddit 上被分享，积累了一批忠实用户，收获了不少流量，但网站运营是有成本的，而功能又是免费的，创始人 David 转变思路，开通打赏功能，解决了运营成本问题，同时也让他有时间去丰富产品功能。

产品运营

产品运营的初期，David主要依靠社交媒体，就比如最开始的Reddit，后来他在 Tiktok 推广产品时发现教学Excel视频的浏览量并不低，这些视频作者的粉丝有很多，于是他意识到了KOL的重要性，开始考虑与Excel相关领域如会计、金融和数据分析的KOL合作，粉丝量少一点的给予免费使用权限，粉丝量大的则付出推广费（简而言之就是请大博主打广告），靠着KOL的影响力，Formulabot 的用户量迎来爆发式增长。

KOL(Key Opinion Leader)：关键意见领袖，指在特定领域内拥有专业知识、影响力和话语权，能通过自身观点和行为影响他人消费决策或观念的人

有时候，酒香也怕巷子深。网页的排序对产品流量也至关重要。在SEO(Search Engine Optimization，搜索引擎优化)方面，David围绕”VLOOKUP“、”SUMIFS“、”Excel嵌套“等特定公式关键字构建专门的内容页面，写教程吸引用户点击，再放上Formulabot的网页链接引流。这种打造高质量外链，间接提升网站排名的方式值得借鉴。

我们可以利用 similarweb 看看Formulabot 目前的流量状况。

我们可以发现：

  1. 2025年8月的流量份额有上涨，8月份每天访问量保持在5000左右，主要市场份额在印度，或许是由于印度的职场人士对Excel处理的需求较大。
  2. 流量份额前5名，要么是搜索引擎，要么是社交媒体，侧面印证了Formulabot 的产品运营策略是社交媒体推广和SEO。

接下来是一些竞品数据对比分析。

可以看到，Formulabot 的每月访问量和页面浏览量最高，用户平均停留时间也最高，说明其用户留存率可能不错，以上数据均说明Formulabot的产品运营策略是成功的。
